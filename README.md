# conflict_practice
clone과 conflict 실습을 위한 레포지토리입니다.

1. 클론을 받는다.
```bash 
git clone URL
```
2. 로컬에서 commit을 하나 더 쌓은 다음에 remote로 push 한다.
3. 리모트에서 commit을 하나 더 쌓은 다음, local에서 pull한다.
4. commit을 쌓습니다.
    - 로컬에서 commit을 쌓습니다.
    - 리모트에서 commit을 쌓습니다.
5. push를 하려고 하였더니 rejected가 뜹니다.
6. 이를 해결하기 위해 pull을 먼저 합니다.
7. merge commit을 남깁니다.
